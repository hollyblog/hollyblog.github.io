欢迎你来到我的个人博客！

我是 Holly，一个还在起步路上的技术小白。

现在的我，正埋头啃着 Java 和 MySQL 的基础语法，一遍遍调试代码，一次次理解数据表之间的关系。我知道自己还很“菜”，但我不怕承认——因为每个大神都曾是从第一行 “Hello, World!” 开始的。

我选择用这个博客记录下学习中的每一个脚印：可能是某个让我卡住三天的 Bug，也可能是一段终于跑通的脚本；是读书笔记，也是实战心得。我不追求高深，只想真实地留下自己成长的痕迹。

我希望自己能一步步走向更广阔的领域：分布式系统、中间件、云计算、大模型……这些词现在听起来还很遥远，但我愿意用时间和努力去靠近它们。

我相信：保持天真，才能对未知保持好奇；保持烂漫，才能在枯燥的代码中找到乐趣。

这里也会分享我放在 GitHub 和 Gitee 上的项目——也许它们现在还很稚嫩，但每一个都是我思考与尝试的结晶。

如果你也正在学习路上，欢迎你常来看看。我们可以一起交流，彼此鼓励。

永远天真，永远烂漫，永远在路上。
